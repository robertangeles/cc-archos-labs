# Archos Labs

AI transformation practice and product studio. Senior data architecture and AI integration consulting, plus AI-powered tools for executive decision-making — starting with the Executive AI Diagnostic.

Built by practitioners. For programs that can't afford to get it wrong.

Live site: [archoslabs.xyz](https://archoslabs.xyz)

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4
- pnpm
- Deployed on Render

## Local development

```bash
pnpm install
pnpm dev
```

App runs at [http://localhost:3007](http://localhost:3007).

## Deploy

`main` is the trunk. Every push auto-deploys via Render. CI must pass before merging.
